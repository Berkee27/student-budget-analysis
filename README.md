# Student Budget Analysis

## Problem

Students often have difficulty understanding where their money goes. This project analyzes student expenses from a CSV file and shows total spending, category-based spending, and payment method distribution.

## Technologies

- Python
- Pandas
- Matplotlib

## How It Works

The project reads expense data from a CSV file, cleans missing values, groups expenses by category and payment method, calculates total spending, and creates visual charts.
